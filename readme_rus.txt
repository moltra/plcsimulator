Сайт программы: http://plcsimulator.wiki.sourceforge.net/

Краткое описание ПО:
Программа (далее - Симулятор) предназначена для симуляции работы промышленного программируемого контроллера (ПЛК), обменивающегося данными через протокол ModBus.
Симулятор может использоваться для отладки SCADA системы, использующей для получения данных - протокол ModBus.

Возможности программы:
Обмен данными по протоколу ModBus как ведомое устройство (Slave)
Обмен данными по протоколу ModBus через интерфейсы RS, Ethernet.
Визуализация и изменение значений размерностью: бит, байт, слово, двойное слово.
Конфигурирование симулируемых тегов в файле Office Excel (.xls)
Моделирование программы ПЛК (индивидуально за вознаграждение)

Ограничения:
На данном этапе развития программы поддерживаются только 03 и 04 функции ModBus.
Накладывается OLE сервером программы Modbus Slave в 2048 тегов.
Количество отображаемых тегов прямо пропорционально увеличению времени отклика симулятора на действия пользователя.

Требования к компьютеру и другому ПО:
ПК удовлетворяющий требованиям для запуска ОС Microsoft Windows XP SP3.
ОС Microsoft Windows (разрабатывалось и тестировалось на XP SP3, работа с другими версиями возможна, но не гарантирована)
Microsoft Office Excel (разрабатывалось и тестировалось на 2003, работа с другими версиями возможна, но не гарантирована)
